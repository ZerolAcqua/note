    注记¶

    支持 attention, caution, danger, error, hint, important, note, tip, and warning

    若要用标题，可以在 attention 等后面加上"标题参数"。如：!!! abstract "标题"

    1. 只有图标

      Abstract

      简介

    2. 可展开

      Abstract

      简介

      ??? abstract
    简介

    3. 默认展开

      Note

      笔记

      ???+ note
    笔记
